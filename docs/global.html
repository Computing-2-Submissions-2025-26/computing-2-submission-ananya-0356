shuffleDeck(cards) → {Array}

Description:

  • Shuffles a deck of hanafuda cards using Fisher-Yates algorithm. This is the first step of the card game. This should happen for when the cards are dealt to the players and the table. Pure function returns new array, doesn't modify input.\

Source:
  • hanafuda.js, line 14
Example
const cards = [card1, card2, ..., card52];
const shuffled = shuffleDeck(cards);
Parameters:
Name Type Description
cards Array

Array of card objects to shuffle
Returns:

New shuffled deck

Type
Array

dealCards(deck) → {Object}

Description:

  • Deals cards from shuffled deck to players and field. Divides 48-card deck into 8 for each player and 8 for the table, leaving 24 remaining deck.

Source:
  • hanafuda.js, line 37
Example
const shuffledDeck = shuffleDeck(allcards);
const game = dealCards(shuffledDeck);
Parameters:
Name Type Description
deck Array

Shuffled deck of cards
Returns:

Contains player1Hand, player2Hand, fieldCards, remainingDeck

Type
Object
